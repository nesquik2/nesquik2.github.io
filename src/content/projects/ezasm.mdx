## Overview
Open-source project to help teach students in Computer Architecure and Organization about concepts in lower level assembly languages.

## Features

- run instruction files, or enter instructions line by line
- lists state of all registers, highlighting changed values
- go forwards, backwards, or pause steps while running instructions

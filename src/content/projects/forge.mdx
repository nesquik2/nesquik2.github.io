## Overview
The new RPI Forge (student makerspace) website

## Features

- Pages: Learn, Status, Account, Hours
- Check the status of all machines at the Forge at a glance
- Check details of all active jobs on machines and job queues
- Admin View: Edit machines, clear and mark failures
- View account details, job history, dues owed
- View tutorials on how to add jobs and use equipment 
- View Forge schedule and events calendar

## My Contributions
- Live Status Page: configured tools for different account roles, live queues, machine status, and created compact UI 
- Prototyped pages
- Fixed critical bugs

## Technologies Used

TypeScript, Python, CSS, HTML, Docker, FastAPI

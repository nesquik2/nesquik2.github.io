## Features

- Collects local news, weather, and air quality data
- Runs sentiment analysis on relevant data, and displayed through deskpet emotions
- Generates sustainbility tips or action items based on news
- Custom graphics!

## Technologies Used

Python, MCP, Gemini

## How we built it

Our app is made with Python, we used the Tkinter UI library and a state machine to display our desktop pet and facilitate user interaction. We used the FastMCP library and Google Generative AI in the backend to generate tailored pet responses based on data from the OpenWeatherAPI and NewsAPI.
